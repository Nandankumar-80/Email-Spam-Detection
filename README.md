# 📧 Email Spam Detection using Machine Learning

This project is a Machine Learning-based Email Spam Detector that classifies messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP) techniques.

---

## 🚀 Features

- Text preprocessing and cleaning
- Feature extraction using CountVectorizer
- Spam classification using Machine Learning
- Real-time message prediction
- High accuracy model

---

## 📂 Dataset

- Dataset used: **SMS Spam Collection Dataset**
- Contains labeled messages as:
  - Spam
  - Ham (Not Spam)

---

## 🛠️ Technologies Used

- Python 🐍
- Pandas
- NumPy
- Scikit-learn

---

## ⚙️ Machine Learning Model

- Algorithm: **Multinomial Naive Bayes**
- Suitable for text classification problems
- Fast and efficient model

---

## 📊 Workflow

1. Data Collection  
2. Data Cleaning  
3. Text Vectorization (CountVectorizer)  
4. Train-Test Split  
5. Model Training  
6. Prediction  
7. Evaluation  

---

## 📈 Results

- Accuracy: ~97% (may vary)
- Successfully classifies spam and non-spam messages
